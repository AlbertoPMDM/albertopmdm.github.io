# Uses

Tools that are part of my stack

## Development

- IDEs: VSCode, Jupyter

- Python, C, SQL

- Git for version control

- Numpy, matplotlib/potly, pandas, scikit-learn

- Dash

    for making dashboards, and surprisingly, it has turned out to be an easy way of making web GUIs for microcontrollers

## Software

- Linux (Kubuntu) and Windows

- Obsidian
    
    for notetaking and organizing research, love the [latex suite](https://github.com/artisticat1/obsidian-latex-suite) plugin for writing math quickly

- KiCAD, LTSpice, Onshape, and Cura (although looking forward to switch to PrusaSlicer)

## Electronics

- ESP32, arduino, and learning STM32

- DSO150 oscilloscope
    
    very cheap oscilloscope, but bandwidth is enough for general debugging and audio work
- Cheap multimeter

    which i very often do not use, preferring the oscilloscope

- TC1 tester
    
    very handy when working with transistors and diodes, and when forgetting resistor band values

- Alientek T80 soldering iron
    
    although not open source to my dislike, it is very capable, and uses nice C245 tips

## Music

- Ableton

- DIY midi controlers

- Roland Aira S-1
    
    despite the menu diving aspect, it is an excellent digital synth

- Boss ME-25 multieffects unit

    On the older side of multi-effects units, but is built to last, and i really like the simple menus instead of the screens newer pedals have. Looking forward to update to the ME-80 or 90.

- Squier Jaguar
